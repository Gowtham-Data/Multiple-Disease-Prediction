# Multiple-Disease-Prediction
Multiple Disease Prediction

This is a simple healthcare prediction app where you can check the possibility of:

✅ Kidney Disease  
✅ Parkinson’s Disease  
✅ Liver Disease  

Just provide a few medical details…  
**Click Predict → You instantly get the result with probability** 

---

### Why this App?
To help people and doctors get a **quick health indication** using Machine Learning.  
This app is **not a medical diagnosis**, but an early warning support tool.

---

###  What I Used
- Python & Streamlit → For building the application
- Machine Learning Models (Scikit-learn)
- Pandas → Data handling
- Joblib → Loading saved models

---

1️⃣ Make sure all required `.pkl` model files are in the same folder
